# FleetFlow SaaS Engine Backend

This is the Node.js backend for the FleetFlow Logistics SaaS, containerized and running on Hugging Face Spaces.

## API Documentation
Once the Space is running, you can access the interactive Swagger documentation at:
`https://mayankg09-fleetflow-backend.hf.space/api-docs`

## Local Development
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file based on the provided secrets
4. Run `npm start`
